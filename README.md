# soru1
C dilinde sıralama algoritmaları ve Trie veri yapısı uygulamaları

Bir X dizisine 0-1000 arası rasgele 500 tamsayı eleman yerleştirip daha sonra bu diziyi hem “Birleştirmeli Sıralama (Insertion Sort)” hem de “Seçmeli Sıralama (Selection Sort)” algoritmaları ile sıralayıp her bir algoritmanın sıralama sürelerini ölçerek ekrana uygun bir mesaj ile gösteren program parçasını C programlama dili ile yazınız.
